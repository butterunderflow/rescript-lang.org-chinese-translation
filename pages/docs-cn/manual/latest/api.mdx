# 概览

ReScript在其标准库中提供了三个模块

- [Js](api/js): 提供全部JavaScript APIs的绑定
- [Belt](api/belt): 提供额外的JavaScript集合（collections）与帮助函数（Helpers）
- [Dom](api/dom): 提供DOM相关的类型（types）和模块（modules）

使用方法:

- 默认使用 `Js` 模块。 其中大部分APIs是无运行时的（runtime-free），并且会编译为清晰、可读的JavaScript代码。
- 对于那些在标准JavaScript中不存在的APIs（当然也不存在于 `Js` 模块中），则可使用Belt。
- `Dom` 模块包含了很多被用于用户空DOM绑定的标准化类型。 但由于DOM本身的复杂性， 您可以不使用该模块，而是为您的应用创建自己的DOM绑定。

**注意**: 不推荐使用其他标准库的替代方案 （除非是DOM绑定）， 因为这可能会导致混乱并分裂社区。
